# Análise Comparativa de Tecnologias para Federação de Identidade

- A metodologia adotada neste trabalho possui caráter experimental e comparativo. Inicialmente, foi conduzida uma análise teórica das tecnologias SAML, OAuth2, OpenID Connect e OpenID Federation, com o objetivo de identificar suas características arquiteturais e formular hipóteses acerca da complexidade de configuração de cada abordagem.

- Na etapa seguinte, foi implementado um ambiente experimental controlado, no qual cada tecnologia foi integrada de forma isolada em um contexto equivalente de federação. Para cada cenário, foram coletadas métricas de desempenho sob duas perspectivas: a do operador da federação e a do usuário final do sistema. Este repositório reúne os ambientes experimentais desenvolvidos, disponibilizando os procedimentos necessários para instalação, configuração e reprodução dos cenários analisados.
