# Comparação entre Protocolos de Federação de Identidade

A metodologia adotada neste trabalho é de natureza experimental e comparativa. Inicialmente, foi realizada uma análise teórica das tecnologias SAML, OAuth2, OpenID Connect e OpenID Federation, com o objetivo de identificar suas características arquiteturais e propor hipóteses sobre a complexidade de configuração.

Em seguida, foi implementado um ambiente experimental controlado, no qual cada tecnologia foi integrada separadamente em um contexto semelhante de federação. Para cada cenário, foram coletadas métricas de desempenho, tanto sob a perspectiva de um operador da federação quanto de um usuário comum do sistema.
